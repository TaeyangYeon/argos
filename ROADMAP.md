# 로드맵

이 문서는 Argos의 향후 개발 방향에 대한 아이디어를 정리합니다.
아래 항목들은 **확정된 계획이 아닌 검토 중인 아이디어**입니다.

---

## v1.1 — EL/DL 모델 연동 (아이디어)

현재 Feasibility 분석에서 EL/DL을 추천만 하고 있으나, 실제 모델 학습/추론을 통합하는 방향.

- [ ] Edge Learning 모델 학습 파이프라인 통합
- [ ] Deep Learning 모델 학습 파이프라인 통합 (YOLO, Anomaly Detection 등)
- [ ] GPU 가속 지원 (CUDA / OpenCL)
- [ ] 모델 학습 진행 상황 UI 표시
- [ ] 학습된 모델 성능 평가 및 비교

---

## v1.2 — 프로젝트 관리 및 배치 처리 (아이디어)

다중 프로젝트 관리와 대량 이미지 처리를 위한 기능 확장.

- [ ] 다중 프로젝트 관리 (프로젝트 생성/삭제/전환)
- [ ] 배치 처리 (여러 이미지 세트 일괄 분석)
- [ ] 분석 결과 이력 관리 및 비교
- [ ] 프로젝트 설정 내보내기/가져오기
- [ ] 원형/다각형 ROI 지원
- [ ] 복합 검사 목적 (하나의 ROI에 여러 검사 조건)

---

## v2.0 — 클라우드 및 협업 (아이디어)

로컬 전용에서 클라우드 지원으로 확장하는 장기 비전.

- [ ] 클라우드 배포 옵션 (웹 기반 인터페이스)
- [ ] 팀 협업 기능 (분석 결과 공유, 코멘트)
- [ ] 실시간 카메라 입력 지원
- [ ] 3D 포인트 클라우드 지원
- [ ] 비전 라이브러리 직접 연동 (Keyence SDK, Cognex VisionPro 등)
- [ ] 자동 리포트 생성 및 이메일 발송
